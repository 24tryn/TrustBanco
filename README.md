TrustBanco - Secure Banking App

A modern, responsive banking app built with HTML5, Tailwind CSS, and Vanilla JS. Includes PIN-based authentication, encrypted storage, transfers, and account management.

🎯 Key Features

• Authentication & Security• 6-digit PIN login, PIN recovery, session timeout
• XOR-based encryption (localStorage)
• Secure PIN change

• Account Management• Profile & security settings
• Dashboard with balance & activity

• Money Transfers• Send funds with validation & confirmation
• Real-time balance updates
• Transaction history
